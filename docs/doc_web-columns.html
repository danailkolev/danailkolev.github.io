WEB How to make colmns

The best way to make columns when making a website without some kind of a builder is to use "flex". This is if you need two equal columns. First you have to add class tags to the divs. We will use "row" and "column" tags to indicate which is which.

Example HTML:

            
div class="row"
  div class="column"
  div class="column"
div

Then you add the CSS to a separate file or in a style tag or in a style attribute.

Example CSS:

            
.row {
  display: flex;
}

.column {
  flex: 50%;
}

And if you want to make more than two or unequal columns the HTML is similar but the CSS is something like this. In the HTML you just have to change the "column" classes to "left", "middle" and "right" or something like that.

Example CSS:

            
.row {
  float: left;
}

.left, .right {
  width: 25%;
}

.middle {
  width: 50%;
}
