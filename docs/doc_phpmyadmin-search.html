SQL How to search for keywords in phpMyAdmin

The best way to search for a keyword in phpMyAdmin is NOT by searching in the filter section above the tables (how I did and lost around 45 min), but the correct way is to:

  1. Click on the "Search" button in the top menu.
  2. Then type the keyword in the top section (you can use %keyword% for a wildcard).
  3. Now you will see which tables contain your keyword.
  4. Click on the "Browse" button next to the table containing your keyword and it will show you only the rows with your keyword.
